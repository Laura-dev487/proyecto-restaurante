# 🍽️ Restaurante Sabor y Arte — Landing Page Conceptual

Una landing page moderna, atractiva e interactiva diseñada para la industria gastronómica. El proyecto está enfocado en destacar la propuesta culinaria del restaurante, presentar la carta de forma clara y guiar al usuario hacia la conversión (reservas y pedidos).

> 📌 **Proyecto de Portfolio:** Esta página fue desarrollada como un concepto de diseño y desarrollo web para demostrar habilidades técnicas en maquetación, interacciones visuales y estructura orientada a resultados.

## ✨ Características destacadas

* **Diseño Responsive:** Totalmente adaptado para una navegación fluida en dispositivos móviles, tablets y computadoras de escritorio.
* **UI/UX Gastronómica:** Uso estratégico del color, tipografía y jerarquía visual para despertar el interés del cliente.
* **Componentes Interactivos:** Carrusel visual para platos destacados y secciones estructuradas para el menú.
* **Detalles Visuales Cheados:** Personalización de componentes de interfaz (como barras de scroll e interacciones en botones) alineados a la identidad visual.
* **Estructura Semántica:** HTML5 claro, limpio y optimizado para accesibilidad.

---

## 🛠️ Tecnologías utilizadas

* **HTML5:** Estructura semántica de la landing page.
* **CSS3:** Flexbox, CSS Grid, animaciones y estilos personalizados.
* **JavaScript:** Interacciones y dinamismo en pantalla.
* **Librerías externas:**
  * [Animate.css](https://animate.style/) — Para efectos y transiciones visuales.
  * [Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/) — Para el carrusel de imágenes/platos.
